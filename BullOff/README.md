## 🛠️ Utilisation dans le Constructeur de Séquences
Dans le module **Séquences**, lorsque vous configurez une séquence :
1. Sélectionnez le **Niveau** (ex: `CAP`, `2nde`, `1ere`, `Terminale`) et la **Matière** (ex: `Maths` ou `Sciences`).
2. Le système chargera dynamiquement le module BO correspondant pour afficher les compétences à valider.
3. Cochez les capacités et connaissances traitées dans votre séquence pour suivre le taux de couverture globale de vos cours par rapport au programme officiel.
