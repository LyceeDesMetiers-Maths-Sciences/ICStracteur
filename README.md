# 🗓️ ICStracteur — Guide d'Utilisation Enseignant

**ICStracteur** est une application locale simple et sécurisée conçue pour vous aider à suivre, organiser et planifier vos cours de l'année à partir de votre emploi du temps Pronote.

> [!IMPORTANT]
> **Respect de la vie privée (RGPD) :** L'application fonctionne entièrement en local dans votre navigateur. Vos données d'emploi du temps, vos annotations (cours faits, documents, absents) ne sont jamais envoyées sur un serveur externe. Tout est stocké uniquement sur votre ordinateur.

---

## 🚀 Démarrage Rapide en 4 Étapes

### Étape 1 : Exporter votre calendrier depuis Pronote
Pour charger vos cours dans l'application, vous devez d'abord récupérer votre fichier d'emploi du temps au format `.ics` (iCalendar) depuis Pronote :
1. Connectez-vous à votre espace **Pronote**.
2. Allez dans votre **Emploi du temps**.
3. Recherchez l'option d'exportation (généralement représentée par une icône de calendrier avec une flèche d'export ou via le menu des paramètres).
4. Exportez et téléchargez le fichier `.ics` sur votre ordinateur.

### Étape 2 : Lancer l'application
L'application ne nécessite aucune installation :
1. Ouvrez le dossier de l'application sur votre ordinateur.
2. Double-cliquez sur le fichier [index.html](file:///home/jeanpat/Documents/Lyc%C3%A9ePro/2026-2027/Projets/WebApp/ICStracteur/index.html).
3. L'application s'ouvre instantanément dans votre navigateur internet habituel.
4. Utilisez le bouton d'importation en haut de la page pour charger votre fichier `.ics` téléchargé à l'étape 1.

### Étape 3 : Sélectionner votre classe
Dans la barre latérale de gauche :
1. Sélectionnez la **Classe** souhaitée (par exemple `_1CAPAS`, `1ASSP1`, etc.).
2. Le cas échéant, sélectionnez le **Groupe** (Classe entière, `G1`, `G2`...). L'application filtrera automatiquement pour n'afficher que les séances qui vous intéressent.

### Étape 4 : Choisir le type de vue
En haut ou à gauche de l'interface, basculez entre les différents modes de visualisation selon vos besoins :
*   **📋 Vue Liste :** Un tableau chronologique de vos cours. Idéal pour cocher rapidement les séances faites, noter les documents distribués ou enregistrer les absents et évaluations.
*   **🖼️ Vue Grille :** Une présentation sous forme de cartes claires, facilitant la lecture globale du contenu de vos séances.
*   **📊 Vue Synthèse (Planification) :** Une vue d'ensemble annuelle sous forme de calendrier mensuel où chaque séance est représentée par un carré de couleur correspondant à sa séquence pédagogique. Parfait pour visualiser la progression annuelle de vos classes de CAP ou Bac Pro (Maths, Sciences, Co-intervention).

---

## 💡 Astuces pratiques
*   **Sauvegarde locale :** Vos annotations (suivi, documents, absents) sont automatiquement conservées dans le navigateur de votre ordinateur. Évitez de vider l'historique complet ou les données de site de votre navigateur pour ne pas perdre votre suivi.
*   **Séquences colorées :** Vous pouvez associer des codes couleur à vos séquences dans la vue Synthèse pour voir en un clin d'œil l'avancée de vos progressions.
